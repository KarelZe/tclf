    Trade classification for python¶

    scikit-learn-compatible implementation of popular trade classification algorithms to classify financial markets transactions into buyer- and seller-initiated trades.

    Algorithms¶

    • Tick test
    • Quote rule
    • LR algorithm
    • EMO rule
    • CLNV rule
    • Depth rule
    • Tradesize rule
